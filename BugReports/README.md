# Bug Reports Folder

This folder contains sample bug reports for the LMS project, documenting issues found during manual testing.

## Contents
- **LMS_BugReports.xlsx** – List of bugs including bug ID, title, steps to reproduce, expected vs actual results, severity, and status.
- **Screenshots/** – Optional folder for screenshots of bugs and error messages.

## How to Use
- Open the Excel or PDF files to review detailed bug reports.
- Each report demonstrates QA practices in documenting, tracking, and prioritizing software issues.
