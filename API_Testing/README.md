# API Testing Folder – Assertion Tests

This folder contains API testing work demonstrating **assertion testing using Postman**.

## Contents
- **Postman_Collections/** – Postman collection JSON files for API endpoints, including:
  - GET requests
  - POST requests
  - Status code verification
  - JSON response validation
  - Response time checks
- **Screenshots/** – Screenshots showing:
  - Request & response details
  - Test results and assertion outputs
  - Response times for performance checks

## How to Use
- Import the Postman collection into Postman to run the tests.
- Open screenshots to see evidence of assertion tests:
  - ✅ Status codes are correct (200, 201, etc.)
  - ✅ JSON fields validated against expected values
  - ✅ Response times measured and recorded
- Demonstrates practical knowledge of **API testing, validations, and QA documentation**.
