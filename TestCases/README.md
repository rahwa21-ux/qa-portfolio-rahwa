# Test Cases Folder

This folder contains all manual test cases for the LMS project, organized by functionality.

## Contents
- **LMS_TestCases.xlsx** – Manual test cases including login, enrollment, and course management.
- **PDF/** – Optional folder for pdf of test cases.

## How to Use
- Open the PDF files to view test steps, expected results, and status.
- These test cases demonstrate QA documentation and best practices.
