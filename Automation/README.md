# Automation Testing Folder

This folder contains automation tests demonstrating **browser automation and assertion checks using Selenium WebDriver**.

## Contents
- **Scripts/** – Selenium scripts automating tasks such as:
  - Opening a browser and navigating to a website
  - Performing a Google search
  - Waiting for specific conditions (e.g., title contains keyword)
  - Closing the browser after the test
  - Assertions: Validating that the page title contains the expected keyword
- **Screenshots/** – Screenshots captured during automated test runs (optional), showing:
  - Browser state before and after actions
  - Verification of test results
  - Any errors encountered
